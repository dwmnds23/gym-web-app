# 🏋️ Προπόνηση — Προσωπικό ημερολόγιο γυμναστηρίου

Web εφαρμογή για το κινητό που καταγράφει προπονήσεις δύναμης, cardio και σωματικό βάρος. Τρέχει offline μετά την πρώτη φόρτωση, αποθηκεύει τα πάντα τοπικά στη συσκευή και προστίθεται στην οθόνη Αφετηρίας σαν κανονική εφαρμογή.

🔗 **Live:** https://dwmnds23.github.io/gym-web-app/

---

## Τι κάνει

- **Προπονήσεις δύναμης** με 4 διαφορετικά προγράμματα σε εναλλαγή:
  - Τύπος Α (στήθος / πλάτη / ώμοι) × Προπόνηση 1–2
  - Τύπος Β (χέρια / πόδια) × Προπόνηση 1–2
  - Αυτόματη πρόταση της επόμενης προπόνησης με σειρά Α1 → Β1 → Α2 → Β2 → ξανά
- **Καταγραφή ανά άσκηση**: σετ, επαναλήψεις, κιλά, και μόνιμη σημείωση (π.χ. «κλίση στο 6»)
- **Σύγκριση με προηγούμενη φορά** δίπλα σε κάθε άσκηση, μαζί με κουμπί «= ίδιο» για γρήγορη συμπλήρωση
- **Πρόοδος ανά άσκηση**: γράφημα εξέλιξης, καλύτερη επίδοση, μεταβολή από την πρώτη φορά
- **Cardio** με επιλογή αθλήματος (Τρέξιμο / Περπάτημα / Κολύμπι / Rowing / Bike), χρόνο και AVHR
- **Σωματικό βάρος** με ιστορικό και γράφημα
- **Ημερολόγιο** όπου κάθε μέρα γυμναστηρίου χρωματίζεται:
  - 🟢 προπόνηση δύναμης
  - 🔵 cardio
  - 🟢🔵 και τα δύο (διχοτομημένη μέρα)
- **Επεξεργασία ασκήσεων**: προσθήκη, μετονομασία, αλλαγή σειράς, διαγραφή
- **Ιστορικό** όλων των καταγραφών

## Αποθήκευση δεδομένων

Όλα μένουν **τοπικά στο τηλέφωνο** μέσω `localStorage` του browser. Δεν στέλνεται τίποτα σε server, δεν υπάρχει cloud, δεν συγχρονίζεται μεταξύ συσκευών.

Τα δεδομένα είναι σε τέσσερα κλειδιά:

| Κλειδί | Περιεχόμενο |
| --- | --- |
| `gymlog.exercises.v1` | Λίστα ασκήσεων ανά προπόνηση και μόνιμες σημειώσεις |
| `gymlog.sessions.v1` | Καταγραφές προπονήσεων δύναμης |
| `gymlog.cardio.v1` | Καταγραφές cardio |
| `gymlog.weight.v1` | Μετρήσεις σωματικού βάρους |

⚠️ Αν καθαριστούν τα δεδομένα του Safari ή διαγραφεί το site από τις ρυθμίσεις του iOS, χάνεται το ιστορικό. Για ασφάλεια υπάρχει **Ρυθμίσεις → Εξαγωγή** που κατεβάζει ένα αρχείο `.json` ως αντίγραφο. Συνιστάται περιοδικό backup (π.χ. μηνιαία).

## Εγκατάσταση στο iPhone

1. Άνοιξε στο **Safari** το link: https://dwmnds23.github.io/gym-web-app/
2. Πάτα το κουμπί Μοιρασιάς και διάλεξε **«Προσθήκη στην οθόνη Αφετηρίας»**
3. Άνοιγέ την πάντα από το εικονίδιο της οθόνης Αφετηρίας για να βλέπεις πάντα τα ίδια δεδομένα

## Ενημέρωση κώδικα

Νέα έκδοση: ανέβασμα του ενημερωμένου `index.html` στο repo (αντικαθιστά το παλιό λόγω ίδιου ονόματος).

Σύντομος δρόμος: https://github.com/dwmnds23/gym-web-app/upload/main → σύρε το αρχείο → **Commit changes**.

Μετά το commit χρειάζονται 1–2 λεπτά να ενημερωθεί το GitHub Pages. Στο κινητό μετά: refresh ή κλείσιμο/άνοιγμα της εφαρμογής. Τα δεδομένα παραμένουν, αλλάζει μόνο ο κώδικας.

## Τεχνικά

- Ένα αρχείο `index.html`, χωρίς build step, χωρίς dependencies
- Vanilla JavaScript / CSS / HTML

- Hosted σε GitHub Pages
- Χωρίς analytics, χωρίς tracking, χωρίς cookies
